# Calculadora de Puntos para La Generala

Esta página web es una calculadora de puntos diseñada para el juego de La Generala. Permite a los jugadores realizar rápidamente los cálculos de puntuación de sus lanzamientos y llevar un seguimiento de los puntajes durante el juego.
